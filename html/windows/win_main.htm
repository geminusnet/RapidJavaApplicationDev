Welcome Windows Users!

This CD is a companion to the book Rapid Java™ Application Development Using Sun™ ONE Studio 4. It contains a number of software packages, each of which you need to install individually. It also includes code examples, solutions to the even numbered exercises, and answers to the review questions.

We hope you enjoy this package and programming with Java!

Getting Started

To get started using this CD, you need to install the software packages. Before you begin, however, make sure your computer meets the following System Requirements.

NOTE: The following instructions assume you are familiar with the Microsoft Windows® Operating System (Windows 98, Windows NT, Windows 2000 or Windows XP).

Please Note: Although the most recent versions of all programs have been included on this CD-ROM, an Internet connection is required in order to install and configure the software successfully. New software releases are frequently being made available. Connecting to the Internet will allow you to download these software updates, link to Web-based help resources, and, if necessary, access documentation specific to your hardware and software configuration.

System Requirements

Operating System

In order to use everything on this CD, you need to be running Windows 2000 Professional (with latest service packs), Windows XP, Windows NT (with service pack 6) or Windows 98, 2nd edition only.

Processor

To use everything, at a minimum you need to have a 500 MHz (minimum) or faster Pentium III / Compatible processor. Sun™ ONE Studio 4, Community Edition (formerly Forte™ for Java™ 4, Community Edition) requires a minimum 500 MHz processor.

Graphics Display Card / Sound Card

A standard graphics display card is adequate.

RAM

To use everything under Microsoft Windows you need a minimum of 256 MB of RAM.

Disk Space

To install everything on the CD, it is recommended that you have at least 400 MB of hard disk space available.

Additional Software

Installing the Software

Now that you have verified that your system meets the minimum requirements for installing the software, you must read the license agreements. Please read the entire text of each of the license agreements and make sure you agree with them. Now you are ready to install. For instructions on how to install each of the software packages, click here.

Examples

Once you have installed all the software, you are ready to begin working through the examples in the book. The examples directory can be viewed by clicking directly on the Examples link on the startup screen or by clicking here.

The source code for examples are in one zip file example.zip. To run the sample, please install it into c:\radft. If you unzip it using WinZip, enter c:\ in the Extract to field and check Use folder names. If you do not have WinZip installed on your computer you can download a free evaluation copy at www.winzip.com. Ê

Reviews

Chapter-by-chapter answers to review questions supplied in the PDF file format, which can be viewed by clicking directly on the Reviews link on the startup screen or by clicking here.

Exercises

The 'exercise' directory contains solutions to even numbered exercises in one zip file evennumberedexercise.zip. The directory can be viewed by clicking directly on the Exercises link on the startup screen or by clicking here. The exercises are in the directory named as exercisex_y for the yth exercise in Chapter x. If you unzip it using WinZip, enter c:\ in the Extract to field and check Use folder names. If you do not have WinZip installed on your computer you can download a free evaluation copy at www.winzip.com.

To run the exercises without modification, unzip evennumberedexercise.zip to c:\radftexercise.
