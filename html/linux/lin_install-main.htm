Click on each software title's button in the left window to read the installation instructions. Even though all the software is distributed on this CD-ROM, you will need to be connected to the Internet in order to access a number of important links in the installation instructions.

Be sure to pay careful attention to the software requirements relative to prerequisite programs that must be installed prior to proceeding with installing the software packages listed in the left window.

Software Environment Requirements

In order to install and use several of the software programs included on this CD-ROM, you must have certain prerequisite software installed. In most cases this software is provided either as a separate software package or as a component of one of the packages. Some of the requirements, however, are not included on the CD.

The following table outlines the dependencies of the various software programs on one another and on other programs. You should read this now to get an overview and re-read it before installing any of the software on this CD-ROM. Come back to this table again to troubleshoot any installation issues that might arise. Be sure read the annotations which provide more details about many of these requirements. In each case, it is highly recommended that you view both programs' homepages to determine what has changed between the pressing of the CD and your purchase of the textbook.
